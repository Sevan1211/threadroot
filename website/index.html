Local-first OSS

Git for your AI agent harness.

Threadroot turns repo-local skills, tools, connections, rules, and memory into portable agent context for Codex, Claude, Cursor, and Copilot. 

threadroot init
threadroot doctor
threadroot context "write tests"
threadroot mcp setup --write

local CLI bridges threadroot connections add aws-dev --provider aws --command aws

No Cloud Required

V1 stores harness state in version-controlled files. Connections use the user's existing local CLI auth; no API keys, tokens, or hosted execution are required.
